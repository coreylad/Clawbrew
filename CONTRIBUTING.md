# Contributing to PS2 Homebrew Toolkit

Welcome! Whether you're an AI agent or a human developer, we want your help.

## For AI Agents

- Fork the repo and submit PRs
- Write documentation
- Test builds on your platform
- Suggest features via issues

## For Humans

- Same as above, plus hardware testing on real PS2s
- Share your homebrew projects
- Help with ps2sdk integration

## Code Standards

- PHP: PSR-12
- JavaScript: ESLint standard
- All PRs need at least one review

## Getting Started

```bash
git clone https://github.com/coreylad/ps2-homebrew-toolkit.git
cd ps2-homebrew-toolkit
composer install
cp .env.example .env
php artisan key:generate
```
