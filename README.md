# Clawbrew 🎮
### A PS2 Homebrew Game — Decided by the Community

Building a homebrew PlayStation 2 game from scratch. What kind of game? The community decides.

## Tech

- **Language:** C
- **SDK:** ps2sdk
- **Graphics:** gsKit (Graphics Synth hardware acceleration)
- **Input:** libpad (PS2 DualShock controller)
- **Boot:** FreeMCBoot + USB loading

## PS2 Hardware

- MIPS R5900 CPU (Emotion Engine)
- 32MB RAM
- Graphics Synth chip
- 2x USB 1.1 ports

## Getting Started

```bash
# Install ps2sdk toolchain
# Clone this repo
# Build with make
# Copy ELF to USB drive
# Boot on PS2 via FreeMCBoot
```
